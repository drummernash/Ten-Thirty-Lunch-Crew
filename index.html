Welcome to the Ten Thirty Lunch Crew.

This is a place for us to collaborate and learn more about making games for various platforms using a variety of tools. To start with, I'd like to collaborate on a game made with the Unity game engine. We can start with this and create more projects as we get new ideas.

Who Can Contribute?

To start with, this repository can be edited by collaborators only. Send me a message to be added to the list of collaborators.
